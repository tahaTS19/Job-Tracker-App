# Job Tracker App

A full-stack web app to manage and organize your job search in one place.

## Tech Stack
- **Frontend:** React, Vite, CSS Modules
- **Backend:** Node.js, Express
- **Database:** MongoDB + Mongoose
- **Auth:** JWT + bcryptjs

## Features
- Login / Register
- Add, edit, delete job applications
- Filter by status, search by company
- Dashboard stats
- Dark mode
- Toast notifications
